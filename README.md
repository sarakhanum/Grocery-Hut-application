🛒 Grocery Hut

Grocery Hut is a full-stack web application built using Django (Backend) and React (Frontend) to manage grocery items efficiently. It provides a simple and interactive interface to perform CRUD operations.

🚀 Features

Add grocery items

Update item details

Delete items

Responsive and user-friendly UI

REST API integration

🛠️ Technologies Used

Python

Django / Django REST Framework

React

SQLite / (Your database if different)

▶️ How to Run
Backend (Django)
cd backend
python manage.py runserver
Frontend (React)
cd frontend
npm install
npm start
📌 Purpose

This project was developed to practice full-stack development using Django and React, focusing on API integration and CRUD operations.
