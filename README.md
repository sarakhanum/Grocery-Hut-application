# Grocery Hut Application

A modern web-based online grocery shopping application developed using **Django** for the backend and **React** for the frontend. Grocery Hut provides customers with a fast, secure, and user-friendly platform to purchase grocery products online, while administrators can efficiently manage products, orders, and users.

## Features

### User Module
- User Registration and Login
- Secure Logout
- Browse Grocery Products
- Search and Filter Products
- Add Items to Cart
- Update Cart Quantity
- Remove Items from Cart
- Checkout and Place Orders
- View Order History

### Admin Module
- Admin Dashboard
- Add / Edit / Delete Products
- Manage Categories
- Manage Inventory
- View Customer Orders
- Update Order Status
- Manage Users

## Technologies Used

- **Frontend:** React, HTML, CSS, Bootstrap, JavaScript
- **Backend:** Django, Django REST Framework
- **Database:** SQLite / MySQL
